# Span 설정 기능 테스트 체크리스트

## 테스트 환경
- OS: Windows 11
- 빌드: `dotnet build src/Span/Span/Span.csproj -p:Platform=x64`
- 실행: Visual Studio F5 (MSIX 패키징 필요)

---

## 섹션 1: 일반 (General)

### TC-01: 언어 변경
- [ ] 설정 > 일반 > 언어 > "English" 선택
- [ ] 재시작 안내 배너가 표시되는지 확인
- [ ] 앱 재시작 후 UI 텍스트가 영어로 변경되었는지 확인
- [ ] "한국어" 선택 → 재시작 → 한국어 UI 확인
- [ ] "日本語" 선택 → 재시작 → 일본어 UI 확인
- [ ] "System Default" 선택 → 시스템 언어로 복귀 확인

### TC-02: 시작 시 동작
- [ ] "마지막 세션 복원" 선택 → 여러 탭 열어두고 → 앱 종료 → 재시작 시 탭 복원 확인
- [ ] "내 PC 홈 열기" 선택 → 재시작 → 홈 화면(드라이브 목록)으로 시작 확인
- [ ] ⚠️ "특정 폴더 열기" — **미구현 예상** (저장만 되고 실제 분기 없음)
- **예상 문제**: StartupBehavior 값이 저장은 되지만 앱 시작 시 이 값을 읽는 분기 코드 없음

### TC-03: 즐겨찾기 트리뷰
- [ ] 설정 > 일반 > 즐겨찾기 트리 토글 ON
- [ ] 설정 창 닫고 사이드바에 즐겨찾기 트리가 표시되는지 확인 (즉시 적용)
- [ ] 트리에서 즐겨찾기 항목 클릭 시 해당 폴더로 이동하는지 확인
- [ ] 토글 OFF → 트리가 즉시 숨겨지는지 확인

### TC-04: 시스템 트레이
- [ ] 설정 > 일반 > 시스템 트레이 최소화 토글 ON
- [ ] 창 닫기 (X 버튼) 클릭 → 앱이 종료되지 않고 트레이로 최소화되는지 확인
- [ ] 트레이 아이콘 클릭/더블클릭으로 창 복원 확인
- [ ] 토글 OFF → 닫기 버튼 클릭 시 앱 완전 종료 확인
- [ ] 멀티 윈도우: 탭 분리된 2번째 창에서는 트레이 최소화가 적용되지 않는지 확인

## 섹션 3: 탐색 (Browsing)

### TC-09: 숨김 항목 표시
- [ ] 숨김 파일이 있는 폴더(예: C:\Users)로 이동
- [ ] 토글 OFF (기본값) → 숨김 파일/폴더가 보이지 않는지 확인
- [ ] 토글 ON → 즉시 숨김 항목이 나타나는지 확인
- [ ] 단축키 Ctrl+H로 토글 동작 확인
- [ ] 토글 OFF → 숨김 항목이 즉시 사라지는지 확인

### TC-10: 파일 확장자 표시
- [ ] 토글 ON (기본값) → 파일명에 확장자(.txt, .exe 등) 표시 확인
- [ ] 토글 OFF → 확장자 숨겨지고 파일명만 표시되는지 확인
- [ ] 즉시 적용 확인 — 밀러/상세/아이콘 뷰 모두 반영
- [ ] 폴더명은 영향받지 않는지 확인

### TC-11: 체크박스 선택
- [ ] 토글 OFF (기본값) → Extended 선택 모드 (Ctrl/Shift+클릭)
- [ ] 토글 ON → 항목 좌측에 체크박스 나타나는지 확인
- [ ] 체크박스로 복수 선택 가능한지 확인
- [ ] 밀러 컬럼, 상세 뷰, 아이콘 뷰 모두 체크박스 표시 확인
- [ ] 토글 OFF → 체크박스 즉시 사라지는지 확인

### TC-12: 밀러 컬럼 클릭 동작
- [ ] "한 번 클릭" (기본값) → 폴더 클릭 시 바로 하위 컬럼 자동 열림 확인
- [ ] "더블 클릭" → 폴더 싱글 클릭 시 선택만, 더블 클릭 시 하위 컬럼 열림 확인
- [ ] 즉시 적용 확인 — 설정 변경 후 바로 동작 변경
- [ ] 상세/아이콘 뷰에서는 항상 더블 클릭 동작인지 확인 (이 설정은 밀러 전용)

### TC-13: 이미지 미리보기 (썸네일)
- [ ] 이미지 파일(.jpg, .png)이 있는 폴더로 이동
- [ ] 토글 ON (기본값) → 이미지 파일에 썸네일 표시 확인
- [ ] 토글 OFF → 썸네일 사라지고 파일 아이콘으로 대체 확인
- [ ] 토글 ON 복귀 → 썸네일 다시 로드 확인
- [ ] 즉시 적용 확인

### TC-14: Quick Look (빠른 미리보기)
- [ ] 토글 ON (기본값) → 밀러 컬럼에서 파일 선택 후 Space 키 → 미리보기 팝업 표시 확인
- [ ] 미리보기 팝업 내용 확인 (이미지, 텍스트 파일 등)
- [ ] Esc 또는 다시 Space → 팝업 닫힘 확인
- [ ] 토글 OFF → Space 키 → 미리보기 안 뜨고 type-ahead 검색으로 동작 확인

### TC-15: 삭제 확인 대화상자
- [ ] 토글 ON (기본값) → 파일 선택 후 Delete 키 → 확인 대화상자 표시
- [ ] "삭제" 클릭 → 파일 휴지통으로 이동 확인
- [ ] "취소" 클릭 → 삭제 취소 확인
- [ ] 토글 OFF → Delete 키 → 확인 없이 즉시 휴지통 이동 확인
- [ ] Shift+Delete (영구 삭제)는 이 토글과 무관하게 항상 확인 표시하는지 확인

### TC-16: 실행 취소 기록 보관
- [ ] "10" 선택 → 파일 11개 이상 연속 삭제 후 Ctrl+Z 11번 시도
- [ ] 10번째까지만 되돌리기 가능한지 확인 (11번째는 불가)
- [ ] "100" 선택 → 더 많은 실행 취소 가능 확인
- [ ] "50" (기본값)으로 복귀

## 설정별 적용 방식 참조표

| 설정 | 적용 시점 | 메커니즘 |
|------|-----------|----------|
| 언어 | 재시작 필요 | App.OnLaunched에서 로드 |
| 시작 동작 | 재시작 | (미구현) |
| 즐겨찾기 트리 | 즉시 | OnSettingChanged → ApplyFavoritesTreeMode |
| 트레이 최소화 | 다음 닫기 시 | Closing 핸들러에서 직접 읽음 |
| 테마 | 즉시 | OnSettingChanged → ApplyTheme |
| 밀도 | 즉시 | OnSettingChanged → ApplyDensity |
| 아이콘 팩 | 재시작 필요 | App.OnLaunched에서 FontFamily 오버라이드 |
| 폰트 | 즉시 | OnSettingChanged → ApplyFontFamily |
| 숨김 파일 | 즉시 | OnSettingChanged → RefreshCurrentView |
| 확장자 표시 | 즉시 | OnSettingChanged → RefreshCurrentView |
| 체크박스 | 즉시 | OnSettingChanged → ApplyMillerCheckboxMode |
| 밀러 클릭 | 즉시 | OnSettingChanged → EnableAutoNavigation 전환 |
| 썸네일 | 즉시 | OnSettingChanged → ToggleThumbnails |
| Quick Look | 즉시 | 키 핸들러에서 매번 직접 읽음 |
| 삭제 확인 | 즉시 | 삭제 핸들러에서 매번 직접 읽음 |
| Undo 기록 | 즉시 | SettingChanged 구독 → MaxHistorySize 변경 |
| 터미널 | 즉시 | 터미널 열기 시 매번 직접 읽음 |
| 셸 확장/개발자/Copilot | 즉시 | 메뉴 빌드 시 매번 직접 읽음 |
| 컨텍스트 메뉴 | 즉시 | 우클릭 핸들러에서 매번 직접 읽음 |
