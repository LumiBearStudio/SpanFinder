## Span Strengths (vs Explorer/Finder)

| Feature | Description |
|---------|-------------|
| **Miller + Details + Icon Unified** | Three view modes in one app |
| **Dual Pane (Split View)** | Neither Explorer nor Finder offers built-in |
| **Per-Tab Independent View Modes** | Different view mode per tab |
| **Windows Shell Context Menu** | COM interop for 3rd party menus |
| **File Operation Undo/Redo** | Copy/Move/Rename/NewFolder all undoable |
| **Action Log** | JSON log of all file operations |
| **Quick Look (Space)** | Finder-style preview on Windows |
| **Tab Tear-Off** | Stable manual drag loop implementation |
| **245 Features** | Comprehensive feature set already |
