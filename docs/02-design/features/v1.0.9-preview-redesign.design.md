### 2.2 밀러 컬럼 ScrollViewer 내 인라인 미리보기

**방식**: 밀러 컬럼의 마지막에 미리보기 "컬럼"을 추가하는 대신, 밀러 컬럼 영역 우측에 미리보기 영역을 분리.

```
┌──────────────────────────────────────────────────┐
│ [밀러 컬럼 ScrollViewer (Width=*)]  [미리보기(Auto)]│
│ [컬럼1] [컬럼2] [컬럼3:파일선택]    [인라인 미리보기]│
└──────────────────────────────────────────────────┘
│                    Git 상태바                       │
└──────────────────────────────────────────────────┘
```

**Grid 구조**:
```xml
<Grid>
    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="*"/>                    <!-- 밀러 컬럼 -->
        <ColumnDefinition x:Name="InlinePreviewSplitter" Width="0"/>
        <ColumnDefinition x:Name="InlinePreviewCol" Width="0"/>  <!-- 미리보기 -->
    </Grid.ColumnDefinitions>
    <Grid.RowDefinitions>
        <RowDefinition Height="*"/>
        <RowDefinition Height="Auto"/>  <!-- Git 상태바 -->
    </Grid.RowDefinitions>
</Grid>
```

### 2.3 인라인 미리보기 표시/숨김 로직

```csharp
private void UpdateInlinePreview(FileSystemViewModel? selectedItem)
{
    if (selectedItem == null || selectedItem is FolderViewModel)
    {
        // 폴더 또는 미선택: 인라인 미리보기 숨김
        InlinePreviewCol.Width = new GridLength(0);
        InlinePreviewSplitter.Width = new GridLength(0);
        return;
    }

    // 파일 선택: 인라인 미리보기 표시
    InlinePreviewSplitter.Width = new GridLength(2, GridUnitType.Pixel);
    InlinePreviewCol.Width = new GridLength(280, GridUnitType.Pixel);
    // 또는 adaptive: GridLength(1, GridUnitType.Star) + MinWidth=200

    InlinePreviewPanel.UpdatePreview(selectedItem);
}
```

{
    if (newMode == ViewMode.MillerColumns)

        // 사이드 패널 숨김 (밀러는 인라인 사용)

### 3.2 HandleQuickLook 변경

```csharp
private void HandleQuickLook(int activeIndex)
{
    var selectedItem = GetSelectedItem(activeIndex);

    if (selectedItem == null || selectedItem is FolderViewModel)
    {
        _quickLookWindow?.Close();
        return;
    }

    if (_quickLookWindow != null && _quickLookWindow.IsOpen)
    {
        _quickLookWindow.Close();  // Toggle off
    }
    else
    {
        _quickLookWindow = new QuickLookWindow();
        _quickLookWindow.UpdateContent(selectedItem);
        _quickLookWindow.Show();
    }
}
```
